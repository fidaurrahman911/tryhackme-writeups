# Vulnversity

## Room Information

- **Room Name:** Vulnversity
- **Difficulty:** Advanced
- **Platform:** TryHackMe
- **Status:** Completed

---

## Objective

Vulnversity simulates a complete penetration testing engagement against a vulnerable Linux web server. The room covers reconnaissance, service enumeration, web application discovery, exploitation, reverse shell access, privilege escalation, and post-exploitation.

---

## Skills Practiced

- Network reconnaissance
- Service enumeration
- Web directory enumeration
- File upload vulnerability exploitation
- Reverse shell establishment
- Linux privilege escalation
- Post-exploitation enumeration

---

## Tools Used

- Nmap
- Gobuster
- Burp Suite
- Netcat
- Linux Terminal

---

## Key Learning Outcomes

- Performed structured reconnaissance before exploitation.
- Enumerated web directories and application functionality.
- Exploited a vulnerable file upload mechanism.
- Established a reverse shell on the target machine.
- Conducted Linux privilege escalation.
- Reinforced the importance of systematic penetration testing methodology.
