# Search Skills

**Difficulty:** Intermediate

**Platform:** TryHackMe

---

# Objective

The goal of this room was to develop practical search techniques used in cybersecurity investigations. The room introduced specialized search engines and threat intelligence platforms that security professionals use for reconnaissance, malware analysis, vulnerability research, and digital investigations.

---

# What I Learned

During this room I learned how to:

- Use Shodan to discover internet-facing devices and exposed services.
- Analyze suspicious files using VirusTotal.
- Perform efficient cybersecurity research using specialized search techniques.
- Identify open ports, web technologies, and operating systems from public data.
- Understand how threat intelligence platforms help defenders investigate malware and malicious infrastructure.

---

# Practical Skills

## Shodan

Used Shodan to investigate an internet-facing web server.

Learned how to identify:

- Open ports
- Running services
- Server software
- SSL information
- Public host information

This demonstrates how attackers and defenders can discover exposed systems connected to the Internet.

---

## VirusTotal

Used VirusTotal to inspect a suspicious executable.

Learned how to:

- Upload or search file hashes
- Review antivirus detections
- Interpret detection ratios
- Investigate malicious files using community intelligence

This helps security analysts quickly determine whether a file is likely to be malicious.
