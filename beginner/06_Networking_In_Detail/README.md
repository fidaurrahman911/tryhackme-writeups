**Difficulty:** Beginner

# Objective

The objective of this room was to develop a deeper understanding of computer networking by exploring how devices communicate across local and wide-area networks. The room covered networking protocols, addressing, switching, routing, and common troubleshooting techniques.

---

# What I Learned

During this room I learned how to:

- Understand network communication between devices.
- Differentiate between MAC addresses and IP addresses.
- Learn how ARP resolves IP addresses to MAC addresses.
- Understand ICMP and Ping operations.
- Explore MAC spoofing and its practical uses.
- Gain a better understanding of network troubleshooting.

---

# Practical Skills

## MAC Addressing

Learned how every network interface possesses a unique MAC address and how switches use these addresses to forward traffic within a local network.

Also explored MAC spoofing and understood its legitimate uses in security testing, privacy, and penetration testing.

---

## ICMP & Ping

Explored how Internet Control Message Protocol (ICMP) enables devices to verify network connectivity.

Used Ping requests to confirm communication between hosts and better understand packet transmission across networks.

---

## Network Communication

Developed a stronger understanding of how data flows between devices through switches, routers, and different network layers.
