**Difficulty:** Beginner

# Objective

The objective of this room was to understand how the Domain Name System (DNS) translates human-readable domain names into IP addresses. The room explored DNS records, the DNS resolution process, and the role DNS plays in modern network communication.

---

# What I Learned

During this room I learned how to:

- Understand the purpose of the Domain Name System (DNS).
- Learn how domain names are resolved into IP addresses.
- Differentiate between common DNS record types.
- Understand recursive and authoritative DNS servers.
- Explore the DNS lookup process from client to server.

---

# Practical Skills

## DNS Resolution

Learned how DNS converts domain names into IP addresses, allowing users to access websites without remembering numerical IP addresses.

---

## DNS Records

Studied several common DNS record types, including:

- A Record
- AAAA Record
- CNAME Record
- MX Record
- TXT Record
- NS Record

Understanding these records provided insight into how domains are configured and managed.

---

## DNS Infrastructure

Explored the roles of recursive resolvers, root servers, TLD servers, and authoritative name servers in completing a DNS lookup.

# Key Takeaways

DNS is one of the most critical services on the Internet. A strong understanding of DNS is essential for cybersecurity professionals because many attacks, investigations, and defensive techniques rely on DNS analysis, monitoring, and troubleshooting.
