# Key Concepts

## Hydra

Hydra is a password auditing tool capable of testing login credentials against numerous services including SSH, FTP, HTTP, RDP, SMB, MySQL, PostgreSQL, Telnet, VNC, and many others.

---

## Dictionary Attack

Learned how Hydra automates password guessing using a predefined wordlist such as `rockyou.txt`.

---

## Command Structure

Understood the basic Hydra syntax including:

- Username selection
- Password wordlists
- Target service
- Parallel threads
- Service modules

---

## Responsible Usage

Hydra should only be used in authorized environments such as TryHackMe labs, Capture The Flag competitions, or systems where explicit permission has been granted.
